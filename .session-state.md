## Macro Rack workflow (Ableton)
Для VST3 плагинов (Arturia Acid V, Drumazon и т.д.) Ableton не показывает все параметры через AbletonOSC.
Решение — Instrument Rack с Macro-ручками:

1. Клик на девайс внутри трека (не на трек!) → Cmd+G
2. Появится Instrument Rack с 8 Macro-ручками
3. ПКМ на Macro 1 → Map to Parameter → ткни в нужную ручку VST
4. ПКМ на Macro 2 → Map to Parameter → другая ручка VST
5. И т.д.
6. В Web UI нажми Scan — увидишь Rack с параметрами Macro 1…8
7. Мапь суставы на эти Macro через Web UI

**Преимущества**: не сбрасывается при перезагрузке, можно маппить снова.

**Normalization**: при создании маппинга из дерева scan передаёт min/max параметров.
Для Macro Rack в Ableton min=0, max=127. Normalization корректно маппит joint 0-1 в 0-127.
